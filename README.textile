h2. Installation

h3. When including jQuery yourself

Just include FooBar.js after you have included jQuery:

<pre>
  <script src="path/to/jquery.js" type="text/javascript"></script>
  <script src="path/to/foo_bar.js" type="text/javascript"></script>
</pre>

*Note*: include @foo_bar.min.js@ for the minified version

h3. When letting FooBar.js include jQuery automatically

Make sure you put the @jquery@ directory in the same directory as the FooBar.js library:

<pre>
  |-foo_bar.js
  |~jquery
    `-core.js
</pre>

Just include FooBar.js:

<pre>
  <script src="path/to/foo_bar.js" type="text/javascript"></script>
</pre>

*Note*: include @foo_bar.min.js@ for the minified version
